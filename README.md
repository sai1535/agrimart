## вљ™пёЏ Installation Guide

### 1. Clone Repository

```bash
git clone https://github.com/sai1535/agrimart.git
```

### 2. Move Project

Copy the project folder into:

```text
C:\xampp\htdocs\
```

### 3. Start XAMPP

Start:

- Apache
- MySQL

### 4. Create Database

Open:

```text
http://localhost/phpmyadmin
```

Create database:

```sql
agrimart
```

### 5. Configure Database

Update database credentials in:

```php
includes/db.php
```

### 6. Run Migration

```bash
php init_db.php
```

or visit:

```text
http://localhost/agrimart/init_db.php
```

```text
http://localhost/agrimart
```
